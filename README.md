# Saveur — Nigerian Soul · Continental Craft

Saveur is a premium restaurant web application built with React and Vite. It features a modern, responsive design that showcases the fusion of Nigerian and Continental cuisine.

## Features

- **Modern UI/UX**: Professional design with a focus on high-quality food photography and elegant typography.
- **Dynamic Menu**: Categorized menu with support for different cuisine types (Nigerian & Continental).
- **Online Ordering**: Full-featured ordering system with cart management and WhatsApp integration for order fulfillment.
- **Dark Mode Support**: Seamless transition between light and dark themes based on user preference or system settings.
- **Responsive Design**: Optimized for desktops, tablets, and mobile devices.
- **Performance Optimized**: Built with Vite for lightning-fast development and optimized production builds.

## Tech Stack

- **Frontend**: React 19, Vite
- **Styling**: Vanilla CSS, Ant Design (UI Components)
- **Icons**: Ant Design Icons
- **Routing**: React Router DOM 7

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Configuration

The restaurant's core information (name, address, social media, etc.) can be configured in `src/config/restaurant.js`.
